# 🌐 rochakcss-docs

Official documentation and demo website for **RochakCSS** — a lightweight, utility-first CSS framework.

---

## 🚀 About RochakCSS

**RochakCSS** is a simple, Tailwind-like utility framework that lets you style elements using `r-*` classes.

- ⚡ No build step  
- 📦 Zero dependencies  
- 🎯 Utility-first approach  
- 🔥 Styles applied dynamically via JavaScript
